# Yail (Yet another interpreted language)
Created out of pure fun. Not for real use.

## Samples:

### Variables

```js
var x = 3;
var y = x + 3;
println(y);
// output: 6
```

### Supports

- [x] Int, bool, char, string, double, null
- [x] '+' and '-' operators
- [x] Console outputs ``print()`` / ``println()``
